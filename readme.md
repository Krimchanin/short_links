## Установка и запуск

Клонируйте репозиторий

```
git clone https://github.com/Krimchanin/short_links.git
```

Установите требуемые библиотеки

```
pip install -r requirements.txt
```

Запустите main.py

```
python main.py
```

Вы запустили проект

```
http://127.0.0.1:80
```
